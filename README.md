# Password-storage
This project is a JavaScript-based password manager system with user authentication and data storage using localStorage. Users can create an account, log in securely, and store multiple password entries linked to their account. The app allows adding, deleting, searching, and clearing saved passwords, along with displaying all stored data for the current user. It also includes password strength checking during signup and multi-page navigation using DOM manipulation. The project demonstrates advanced use of arrays, objects, localStorage, event handling, functions, and user authentication logic in JavaScript.
